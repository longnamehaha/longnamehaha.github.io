图像读取

其中OpenCV是核心，负责把「图片里的峰」变成「计算机能识别的数字（角度）」。

api

API 函数 作用 对应XRD处理目标
cv2.imread() 读取图片为数字矩阵 加载XRD图谱
cv2.threshold() 二值化，分离峰和背景 提纯衍射峰信号

cv2.boundingRect() 计算峰的外接矩形，获取中心坐标 找到峰的精确位置

json格式

JSON 是轻量级、跨语言、纯文本的数据交换格式，用来存数据、传数据

基本特点

  • 纯文本，人类可读
  • 键值对结构
  • 无注释、无变量、无函数
  • 严格区分大小写
  • 后缀：.json

支持的数据类型

  1. 对象 { }
  2. 数组 [ ]
  3. 字符串（必须双引号）
  4. 数字（整数/浮点数）
  5. 布尔 true / false
  6. 空值 null

语法规则

  • 键必须用双引号
  • 字符串必须双引号
  • 只允许：{ } [ ] , : " " 这些符号

XRR了解

XRR是一种非破坏性的表面与界面分析技术，核心基于X射线在材料表面及界面的反射、折射与干涉现象，与XRD的衍射原理不同，可分析结晶与非结晶薄膜的厚度、密度、表面/界面粗糙度等纳米级参数。

特殊折射率

  • X射线是波长极短（约0.1-10 nm）的电磁波，与物质相互作用主要通过电子散射
  • 关键特性：对X射线而言，绝大多数材料的折射率n略小于1（n=1-δ-iβ，δ、β为小正数），与可见光相反
  • 这一特性导致X射线在材料表面可能发生全外反射（类似光从光密到光疏介质的全反射）

临界角与全反射

  • 临界角θc：当X射线以极小掠入射角（通常<5°）入射时，存在一个临界角度
    • 入射角θ < θc：几乎100%全反射，极少X射线进入材料内部
    • 入射角θ > θc：部分X射线反射，部分透射进入材料并在内部界面反射
  • 临界角公式（近似）：θc ≈ √(2δ) ≈ √(λ²·re·ρ·NA·Z/A·π)，与材料电子密度ρ直接相关
    • λ：X射线波长，re：经典电子半径，NA：阿伏伽德罗常数，Z/A：原子序数/原子量比

Kiessig条纹形成

当θ > θc时，透射的X射线会在多层薄膜的各个界面（如空气-薄膜、薄膜-衬底）发生反射，这些反射波因光程差产生相干干涉：

  • 相长干涉：反射波叠加增强，反射率升高
  • 相消干涉：反射波相互抵消，反射率降低
  • 形成周期性振荡曲线，称为Kiessig条纹
  • 条纹的周期与薄膜厚度d相关（d ∝ λ/(2Δθ·sinθ)），振幅衰减反映界面粗糙度

测量原理与数据分析

  1. 实验方法：以θ/2θ模式测量，保持反射角=入射角，扫描θ角同时记录反射强度
  2. 数据解析：将实验曲线与理论模型（如Parratt递归法）拟合，提取关键参数
    • 厚度：由Kiessig条纹周期计算
    • 密度：由临界角位置推导
    • 粗糙度：由条纹振幅衰减程度判断

XRR与XRD

特性 XRR XRD
原理 表面/界面的反射与干涉 晶体内部原子面的布拉格衍射
适用样品 薄膜（厚度1-1000 nm），结晶/非结晶均可 晶体材料，块体/厚膜为主
信息来源 电子密度分布（界面、密度） 晶体结构（晶格参数、对称性）

小结

XRR利用X射线折射率小于1的特性，通过测量反射率随掠入射角的变化，结合全反射与干涉现象，实现对薄膜纳米级结构的高精度表征。其物理本质是经典电磁理论在X射线波段的特殊表现，通过现代拟合算法可定量获取薄膜的关键参数，广泛应用于材料科学、半导体、光学涂层等领域。
