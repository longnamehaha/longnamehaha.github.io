物理基础

X射线衍射（X-ray Diffraction，XRD），是一种快速、准确、高效的材料无损检测技术。作为一种表征晶体结构及其变化规律的手段，其应用遍及材料、化学、生物、医药、陶瓷、冶金、矿产等诸多领域。

X射线

X射线的本质是一种波长极短（约为10-8 ~10-12 m），能量很大的电磁波，它具有波粒二象性。

由于X射线具有极大的能量，当其到晶体中时，晶体中的原子会在X射线的作用下被迫做周期性的运动，从而会以原子球为单位对外发射次生波，该波的频率与入射X射线一致，这个过程就成为X射线的散射。考虑到晶体中的原子在空间上呈周期性的规律排布，这些散射球面波之间存在着固定的位相关系，会在空间产生干涉，结果导致在某些散射方向的球面波相互加强，而在某些方向上相互抵消，从而也就出现衍射现象。
因此，晶体中的X射线衍射实质上就是大量原子散射波在空间上相互干涉的结果。

X射线衍射与材料

XRD 实验中核心的两类晶体 / 晶态物质

类型 核心作用 晶态要求 大学物理实验常用实例
待测样品晶体 作为 XRD 的研究对象，X 射线照射其晶面发生相干衍射，通过衍射图谱分析其晶体结构 晶态即可（单晶 / 多晶 / 粉末皆可，大学物理实验以粉末晶体为主，易制备、图谱特征明显） NaCl、Si、KCl 粉末
X 射线源靶材晶体 作为 X 射线的发射源，高压下电子轰击靶材晶体，使其产生特征 X 射线（波长固定，是 XRD 的“探测光”） 金属单晶 / 多晶（原子排列规则，才能产生波长固定的特征 X 射线） Cu、Mo、Fe 金属靶

晶体与衍射峰

对于非晶体材料，由于其结构不存在晶体结构中原子排列的长程有序，只是在几个原子范围内存在着短程有序，故非晶体材料的XRD图谱为一些漫散射馒头峰。然而，对于晶体材料，其原子排布在三维空间上长程有序，其XRD衍射图谱只在特定的位置上出现加强峰（X射线衍射加强结果）。晶体产生的衍射花样都反映出晶体内部的原子分布规律。

概括地讲，一个衍射花样的特征，可以认为由两个方面的内容组成:

  1. 衍射线在空间的分布规律—由晶胞的大小、形状和位向决定;
  2. 衍射线束的强度—取决于原子的种类和它们在晶胞中的位置。

衍射花纹就像晶体的指纹一样，通过鉴别衍射花纹的在空间上出现的位置和强度，即可在X射线衍射和晶体结构之间建立定性和定量关系。

相关公式

布拉格方程

衍射分析中最重要的基础公式，是XRD理论的基石。它简单明确地阐明衍射的基本内涵，揭示了衍射与晶体结构的内在关系。
$$
nλ=2dsinθ, (n=1, 2, 3…..)
$$
其中，λ, d, θ分别代表了X射线的波长，晶体晶面间距，入射X射线与相应晶面的夹角。
通过布拉格方程，可以用已知波长的X射线去求解晶体晶面间距d，从而获得晶体结构信息，这就是结构分析；也可以用已知晶面间距的晶体来测量未知X射线的波长，这就是X射线光谱学。

谢乐公式

Scherrer公式是XRD测晶粒度的理论基础。它主要描述了晶粒尺寸与衍射峰半峰宽之间的关系。晶粒越小，XRD衍射线的峰就越弥散宽化；反之则越集中。
$$
D=\frac{Kλ}{Bcosθ}
$$
其中D, K, λ, B, θ分别为晶粒垂直于晶面方向的平均厚度、Scherrer常数、X射线波长、实测样品衍射峰半高宽度（弧度）以及衍射角。Scherrer常数K的值一般由B来决定，当B为衍射峰半高宽，K=0.89；当K为衍射峰面积积分半高宽时，K=1。由于材料中的晶粒大小并不完全一样，故该方法计算的是不同大小晶粒的平均尺寸。

模拟数据说明

  • NaCl ：最经典的待测粉末样品晶体（面心立方结构）
    NaCl 是离子型面心立方晶体，晶面间距大、衍射峰少且特征明显、粉末易制备，是大学物理 XRD 实验的标配待测样品，学生能快速通过其图谱理解 “晶面→衍射峰” 的对应关系；
    实验中形态：粉末状（200-300 目），装在样品台后，无数细小晶粒随机取向，能满足 X 射线在不同晶面的衍射条件，产生完整的衍射图谱；
    在我们代码中的体现：d_list = [0.282, 0.200, 0.141]是 NaCl 不同晶面的晶面间距，代码计算的衍射峰都是 NaCl 晶体的晶面衍射结果。
  • Cu ：X 射线源的 Cu 靶（金属面心立方晶体）
    高压下电子轰击 Cu 金属晶体，Cu 原子的内层电子被激发，外层电子跃迁填补空位时，会释放出波长固定的特征 X 射线（Cu Kα 线，λ=0.154nm）—— 这是 XRD 实验的 “探测光”，波长固定是布拉格方程计算峰位的前提；
    Cu 靶产生的特征 X 射线波长适中，能让大多数常见晶体（如 NaCl、Si）在 5°~80° 的扫描范围内产生明显的衍射峰，且 Cu 金属熔点高、稳定性好，是 XRD 实验的标配靶材；
    在代码中，设定：lambda_x = 0.154nm为 Cu 靶特征 X 射线的固定波长，代码所有峰位计算都基于此值。
