时间序列模型

简单概念

从专业定义上来讲：时间序列是按时间顺序排列的数据集合。这些数据可以在任何时间间隔（例如，每秒、每分钟、每小时、每天、每月、每年等）观察和收集。
时间序列可以简单理解为一串与时间相关的数字，比如每月的销售额，每年的GDP，每天的气温等。

时间序列分类

时间序列分为两类：

  • 时期时间序列：结果表示一段时间总和，比如国家今年GDP，就是这一年的总和。
  • 时点时间序列：结果表示目标某一个时刻达到的值，比如今天的温度，这个月你的体重。

简单的区分方法：时期时间序列的累加是有意义的，比如2024年和2025年GDP累加，就表示这两年的GDP之和；而时点时间序列的累加是没有意义的，比如今天的气温+明天的气温，或者你这个月的体重+上个月的体重，这些值没有意义。

单变量和多变量

单变量时间序列只涉及一个变量随时间的变化。比如上面的每月的销售额变化，只看了销售额这一个变量。
既然有单变量，就会有多变量。多变量时间序列描述一个事件多个指标随时间的变化，而且由于是同一个事件，各个指标之间也会有关系。
比如出售商品，涉及到销售额，广告费，产品数量等指标，这些指标之间会产生联系。

从预测结果的角度，单变量的预测只需要考虑该变量前面时间的数据；而多变量中，每一个变量的预测都需要考虑前面时间每个变量的数据，并不是简单的多个单变量组合起来。

时间序列与机器学习

从前面可以了解到，时间序列就是通过过去数据来预测未来数据，容易联想到机器学习。机器学习也是从已知的数据训练出模型去预测未知的数据。这样看来，是不是用机器学习就可以解除时间序列的预测问题呢？

答案是否定的。

机器学习和时间序列模型的内在不同，

  • 对于机器学习，其训练数据直接往往没有联系，只是单纯组合并训练出模型结果。
  • 对于时间序列模型，其数据是严格依赖时间的，因此无法忽略数据之间的联系，其数据之间存在相互影响，需要额外考虑。

这里我将机器学习和时间序列模型放在同等地位理解，下面的内容也和机器学习过程有很多相似。

时间序列模型建模过程

时间序列模型也是预测未知数据的模型，各步骤类似机器学习。

划分数据集

前面说过，时间序列相关模型，本质任务就是使用过去的数据预测未来的数据。所以训练集一定是用历史数据，而测试集要预测的时间一定在训练集之后。

训练方式

单步预测

仅预测需要的某一个时间点的值，比如通过1月-8月的销售量，预测9月的。
这里，我们只关心这一个点的预测结果，而不考虑后面的结果。

多步预测

预测需要的某一段时间的值，比如通过1月-8月的销售量，预测9月-12月的。
这里，我们不只关心一个点的预测结果。很明显，离现在越远的未来越难预测，因此应该要尽量压缩训练集与测试集之间的时间差。
一种预测方法是按照时间进行，将先预测到的值作为后面预测的参考。比如使用6-8月预测出9月的值，然后使用7-8月的真实值+9月的预测值预测出10月的，以此类推。
当然，对于这样的预测结果，缺陷也是很明显的，因为误差会逐步累计。如果第一步预测有问题，那么后面误差会越来越大。

逐点预测与分布预测

逐点预测是指在需要的时间点，预测一个最可能的值。这是我们最常见的预测形式，例如在天气预测中，我们可能会预测明天的最高气温为28度。
分布预测，顾名思义，即预测需要的点，在该点各个结果可能出现的分别情况。例如在天气预测中，我们可能会预测明天的最高气温为27-29度的可能性为70%，25-26以及30-31的可能性为10%等。

时序任务分类

Forecasting

即前面常举例的预测任务，这里的数据一般需要满足：

  • 数据样本由时间顺序排列
  • 各个样本之间的时间间隔保持一致
  • 数据样本之间存在关联

  • 过去的样本数据影响未来的预测值

    Time Supervision

    类似机器学习，
    时序有监督学习包括时间序列回归和时间序列分类两种有监督学习方法（Time Series Regression/ Time Series Classification），指的是在多变量时间序列上，基于时间或时间相关的数据，完成回归或分类任务，这类任务对样本与样本之间的时间顺序、相关等都没有要求、反而更关注特征之间的关系，因此这类任务要求：

  • 数据中必须存在除时间之外的其他特征，且特征越多越好

  • 可以完成“去时序化”处理，将时序数据彻底变为一般机器学习数据
  • 这一类任务是学习特征值、预测标签值，并不关心样本。

AR模型

基本概念

AR模型，即自回归模型（Autoregressive Model），是统计学中处理时间序列的常用方法，广泛应用于经济学、信息学、自然现象预测及计算机视觉等领域。

重要理解

AR模型当前的结果一定依赖于过去的积累，因此AR模型相信：一个时间点上的标签值一定是依赖于之前的时间点上的标签值而存在的。

MA模型

概念

MA模型，即移动平均模型，是时间序列分析中的一种模型，它描述的是当前时间点的数据与过去噪声的关系。
严格定义上来讲：其模型的定义是基于白噪声序列的假设。白噪声是一种特殊的时间序列模型，每个时间点的数据都是独立且服从相同分布的，且具有常数的均值和方差。

    重要理解

    从定义上来看：MA模型的基本思想和基本假设与AR模型大不相同。MA模型的基本思想是：大部分时候时间序列应当是相对稳定的。在稳定的基础上，每个时间点上的标签值受过去一段时间内、不可预料的各种偶然事件影响而波动。
    也就是说，AR模型是考虑了数据从历史以来的稳定性，而MA模型考虑这种稳定性上的突发情况。因此，二者相辅相成，组成ARMA模型。

ARMA(p,q) 模型（自回归移动平均模型）

ARIMA模型

组成

ARIMA模型全称为自回归差分移动平均模型（Autoregressive Integrated Moving Average Model）。ARIMA模型主要由三部分构成，分别为自回归模型（AR）、差分过程（I）和移动平均模型（MA）。

  • 使用差分将时间序列变平稳
  • 使用AR考虑过去历史值的影响
  • 使用MA考虑过去误差，突发事件的影响

在ARIMA模型（$ARIMA(p,d,q)$）中，参数选择的核心逻辑是 “先定d（差分阶数）→ 再选p（AR阶数）和q（MA阶数）”，其中：

  • $d$ 依赖 ADF平稳性检验（确保序列平稳）；
  • $p$ 和 $q$ 结合 ACF/PACF图形特征（直观判断）和 AIC/BIC准则（客观择优）；
  • 最终通过 残差检验 验证模型合理性。

差分

1阶差分：$\nabla Xt = X_t - X{t-1}$
d阶差分：$\nabla^d X_t = \nabla(\nabla^{d-1} X_t)$

一个序列要平稳，关键是「统计特性不随时间变化」，核心需要满足：

  1. 均值恒定：不管在哪个时间点（比如 t=10 或 t=100），序列的平均水平不变；
  2. 方差恒定：序列的波动幅度（离散程度）不随时间变大或变小；

于是，对于各种非平稳的序列，比如线性增长或者二次增长等，就可以通过一次或多次差分使序列变平稳，序列平稳也是AR和MA模型使用的前提。

ACF与PACF

概念铺垫

  1. 时间序列平稳性：以下所有公式均基于「宽平稳序列」（均值恒定、方差恒定、自协方差仅与时间间隔有关）——非平稳序列需先差分（如ARIMA的d步），再对差分后的平稳序列分析ACF/PACF；
  2. 滞后阶数：用 $k$ 表示「时间间隔」（ $k=0,1,2,…$ ），$k=0$表示序列与自身的相关（恒为1），$k=1$表示 $yt$ 与 $y{t-1}$ 的相关，$k=2$ 表示 $yt$ 与 $y{t-2}$ 的相关，以此类推。

1. 核心定义

1. 核心定义

ACF与PACF的区别

简单理解

  • 假设序列是 $yt$ （今天的气温）、$y{t-1}$（昨天的气温）、$y_{t-2}$（前天的气温）；
  • ACF($k=2$)： $\rho_2$ 衡量今天和前天的「总相关」——包含“前天→昨天→今天”的间接相关（前天影响昨天，昨天再影响今天）和“前天→今天”的直接相关；
  • PACF($k=2$)： $\phi_{22}$ 衡量今天和前天的「直接相关」——剔除了昨天的干扰，只看前天是否直接影响今天（比如前天的冷空气是否直接持续到今天，不通过昨天传递）。

ADF检验

$d$ 是使非平稳序列变为平稳序列所需的 最小差分次数，核心工具是 ADF检验（Augmented Dickey-Fuller Test）。

核心原理

ADF检验的目的是 检验序列是否存在单位根：

  • 原假设 $H_0$：序列存在单位根 → 非平稳；
  • 备择假设 $H_1$：序列不存在单位根 → 平稳；
  • 判定规则：计算ADF统计量，与临界值（1%/5%/10%）对比：
    • 若 ADF统计量 < 临界值 → 拒绝 $H_0$ → 序列平稳；
    • 若 ADF统计量 ≥ 临界值 → 接受 $H_0$ → 序列非平稳，需差分。

步骤流程

  1. 对原始序列 $y_t$ 做ADF检验：
    • 若平稳（ADF统计量 < 5%临界值）→ $d=0$；
    • 若不平稳 → 做1阶差分（ $d^1t = y_t - y{t-1}$ ），对 $d^1$ 做ADF检验；
  2. 对1阶差分序列 $d^1$ 做ADF检验：
    • 若平稳 → $d=1$；
    • 若不平稳 → 做2阶差分（$d^2t = d^1_t - d^1{t-1}$），对 $d^2_t$ 做ADF检验；
  3. 对2阶差分序列 $d^2_t$ 做ADF检验：
    • 若平稳 → $d=2$；
    • 若仍不平稳 → 优先检查序列是否有季节性（需用SARIMA模型），而非继续增加 $d$（$d≥3$ 时模型易过拟合，现实中极少用）。

注意事项

  • $d$ 通常取0、1、2，超过2时模型解释性差，且易出现过拟合；
  • 若原始序列本身平稳（如白噪声），则 $d=0$（此时ARIMA退化为ARMA模型）；
  • 差分过度会导致序列信息丢失（如2阶差分可能把平稳序列变成“过度差分”，方差变大），因此需用ADF检验严格验证，而非盲目差分。

确定 p 和 q：AR阶数和MA阶数

确定 $d$ 后，基于 差分后的平稳序列（记为 $y_{diff}^d$），通过“ACF/PACF图形特征”初步判断 $p$ 和 $q$，再用“AIC/BIC准则”客观择优。

ACF/PACF图形直观判断（快速定范围）

ACF（自相关函数）反映“总相关”，PACF（偏自相关函数）反映“直接相关”，两者的 截尾/拖尾特征 是选择 $p$ 和 $q$ 的核心依据（仅适用于平稳序列）：

模型类型 ACF（自相关函数） PACF（偏自相关函数） 核心逻辑
AR(p) 拖尾（缓慢衰减至0） 截尾（$k=p$后落入置信区间） PACF“截尾”对应AR的阶数p：比如PACF在k=2后显著为0，说明p=2
MA(q) 截尾（$k=q$后落入置信区间） 拖尾（缓慢衰减至0） ACF“截尾”对应MA的阶数q：比如ACF在k=3后显著为0，说明q=3
ARMA(p,q) 拖尾 拖尾 无法仅通过图形确定，需结合AIC/BIC遍历搜索

其实实际问题，通常采用逐个检验的傻瓜方法来确定p和q，通常都是较小的数字。

AIC和BIC

在 ARIMA 模型参数选择中，AIC（赤池信息准则）和 BIC（贝叶斯信息准则）是客观判断最优模型的核心工具—— 本质是通过 “平衡模型拟合效果” 和 “模型复杂度”，避免欠拟合或过拟合，最终选出 “性价比最高” 的模型。

在ARIMA模型参数选择中，AIC（赤池信息准则）和BIC（贝叶斯信息准则）是客观判断最优模型的核心工具——本质是通过“平衡模型拟合效果”和“模型复杂度”，避免欠拟合或过拟合，最终选出“性价比最高”的模型。以下从「核心作用、公式含义、判断规则、关键差异、实操步骤」全维度讲解，结合ARIMA场景落地：

核心作用：解决“主观判断的不足”

之前用ACF/PACF图判断p、q时，“拖尾/截尾”的界定有主观性（比如有人认为k=2截尾，有人认为k=3）；而AIC/BIC通过量化指标，从“所有候选模型”中选出最优，核心目标是：

在“拟合得好”和“参数少（简单）”之间找最优解——既不因为参数太少导致欠拟合，也不因为参数太多导致过拟合。

公式与核心含义

AIC和BIC的公式都围绕“似然值（拟合度）”和“参数个数（复杂度）”展开，先记住核心公式和每个符号的意义：

准则 公式（简化版） 各部分含义
AIC $AIC = 2k - 2\ln(L)$ - $k$：模型的参数个数（ARIMA中$k = p + q + 1$，+1是常数项；比如ARIMA(2,1,1)的$k=2+1+1=4$）
- $\ln(L)$：模型的对数似然值（$L$是似然值，越大表示模型对数据的拟合效果越好；$\ln(L)$越大，$-2\ln(L)$越小）
- $2k$：复杂度惩罚项（参数越多，惩罚越重）
BIC $BIC = k\ln(n) - 2\ln(L)$ - $k$：同AIC（参数个数）
- $n$：样本量（数据行数）
- $\ln(n)$：样本量越大，对参数的惩罚越重
- $-2\ln(L)$：同AIC（拟合度项）

公式的核心逻辑：

  • 拟合度越好（$\ln(L)$越大）→ $-2\ln(L)$越小 → AIC/BIC越小（越优）；
  • 参数越多（$k$越大）→ 惩罚项（$2k$或$k\ln(n)$）越大 → AIC/BIC越大（越差）；
  • AIC/BIC的最终值，是“拟合度”和“复杂度”的综合得分——得分越低，模型越优。

AIC vs BIC：

对比

两者的核心区别是对“模型复杂度（参数个数）”的惩罚力度不同，直接决定了适用场景：

特征 AIC（赤池信息准则） BIC（贝叶斯信息准则）
惩罚力度 轻（惩罚项是$2k$，与样本量无关） 重（惩罚项是$k\ln(n)$，样本量越大，惩罚越重）
核心倾向 优先保证拟合效果，允许稍多参数（易选复杂模型） 优先保证模型简单，严控参数个数（易选简单模型）
适用场景 1. 样本量小（n<100）
2. 希望模型拟合精度高
3. 允许少量过拟合		 1. 样本量大（n≥100）
2. 追求模型解释性（参数少）
3. 避免过拟合
极端情况 样本量再大，也可能选参数多的模型 样本量足够大时，一定会选参数最少的“有效模型”

实操选择技巧：

  • 若AIC和BIC指向同一个模型 → 直接选这个模型（最优解）；
  • 若AIC和BIC指向不同模型：
    • 样本量小（比如n<50）→ 选AIC最优（避免欠拟合）；
    • 样本量大（比如n≥100）→ 选BIC最优（避免过拟合，模型更简洁）；
    • 折中方案：选两个模型中“参数更少、AIC/BIC接近最小值”的（比如AIC最小是162（p=1,q=1），次小是163（p=1,q=0），选p=1,q=0）。

常见误区

  1. 误区1：AIC/BIC越小越好 → 错！如果参数搜索范围过大（比如p=0~10），可能选到过拟合模型（AIC小但泛化能力差），需结合ACF/PACF限定搜索范围（通常0~3/0~4）；
  2. 误区2：只看AIC/BIC，忽略残差检验 → 错！AIC/BIC是“择优”，但不能保证模型“有效”（比如序列有季节性，ARIMA本身不适用，AIC/BIC再小也没用）；
  3. 误区3：不同模型的AIC/BIC可直接对比 → 错！只有“同一数据集、同一d值”的ARIMA模型，AIC/BIC才可比（比如d=1的模型和d=2的模型，AIC不可比）；
  4. 误区4：BIC一定比AIC好 → 错！样本量小时，BIC的强惩罚可能导致欠拟合（比如明明p=2更好，BIC却选p=1），需结合场景。

小结

AIC和BIC的核心判断逻辑是：在限定的p/q范围内，选准则值最小的模型，其中：

  • AIC：“拟合优先”，适合小样本；
  • BIC：“简洁优先”，适合大样本；
  • 最终必须用残差检验验证，避免“最优但无效”的模型。

这两个准则的价值是“把主观判断变成客观量化”，是ARIMA参数选择中最核心的“兜底工具”——结合ACF/PACF的直观判断+AIC/BIC的客观择优，才能选出最优的p、q、d。

ARIMA模型小结

步骤 核心工具 目标 输出
1 ADF检验 使序列平稳 确定d
2 ACF/PACF图 初步判断p和q的范围 p_range, q_range
3 AIC/BIC准则 客观选择最优p和q 确定p, q
4 残差检验 验证模型合理性 确认参数或调整

拓展—SARIMA

SARIMA(p,d,q)(P,D,Q)s 模型（季节性自回归积分移动平均模型）
该模型就是将ARIMA的差分增加滞后性，比如每周的数据，可以进行滞后7的差分，年度每月数据进行滞后12的差分等。其他方面和ARIMA相似。

滞后差分定义

1阶季节性差分（周期s）：$\nablas X_t = X_t - X{t-s}$
D阶季节性差分：$\nabla_s^D X_t = \nabla_s(\nabla_s^{D-1} X_t)$

核心公式

$\nabla^d \nablas^D X_t = \phi_0 + \sum{i=1}^p \phii \nabla^d \nabla_s^D X{t-i} + \sum{j=1}^P \Phi_j \nabla^d \nabla_s^D X{t-js} + \varepsilont - \sum{k=1}^q \thetak \varepsilon{t-k} - \sum{l=1}^Q \Theta_l \varepsilon{t-ls}$
