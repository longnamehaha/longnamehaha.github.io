联合国17个可持续发展目标的优先级排序与发展预测

美赛D题优秀论文拆解：SDGs优先级建模全流程笔记

2023年美赛D题聚焦联合国17个可持续发展目标（SDGs）的优先级排序与发展预测，这篇优秀论文通过三大核心模型构建了完整的建模体系，逻辑严密且方法创新。本文将详细拆解论文的核心流程、原理与关键步骤，为数学建模爱好者提供参考。

一、核心问题与整体框架

1. 赛题核心需求

SDGs并非孤立存在，存在协同关系（相互促进）和权衡关系（资源竞争），论文需解决5大核心问题：

  • 构建SDGs间的关系网络（量化目标间关联）；
  • 制定SDGs的优先级排序标准；
  • 基于优先级预测未来10年SDGs发展；
  • 量化COVID-19等突发事件的影响；
  • 拓展模型在企业/组织中的应用。

2. 整体建模框架

论文通过“3大核心模型+2类扩展分析”形成闭环，逻辑衔接如下：

  1. 数据预处理 → 关系图模型（揭示SDGs关联）
  2. 关系图模型 → 优先级模型（多指标排序）
  3. 优先级模型 → 时间香槟塔模型（未来10年预测）
  4. 时间香槟塔模型 → 突发事件分析（COVID-19影响）
  5. 时间香槟塔模型 → 敏感性分析（验证鲁棒性）

核心逻辑：先量化SDGs的“关联强度”，再基于关联确定“优先级”，最后按优先级和关联动态预测“发展路径”，并验证模型可靠性。

二、数据处理：建模的基础准备

数据是建模的核心支撑，论文的数据分析流程规范且严谨，为后续模型提供了可靠输入。

1. 数据选择

  • 数据源：联合国全球SDG数据库（涵盖17个目标的指标与序列数据）；
  • 样本选择：土耳其、墨西哥、印尼（代表三大洲，覆盖不同发展阶段）；
  • 时间范围：2003-2018年（平衡样本长度与数据完整性）；
  • 序列筛选：共选取95个序列（确保覆盖所有SDGs，兼顾样本量与代表性）。

2. 数据清洗

针对数据缺失问题，采用线性回归填补，公式如下：

b=n∑i=1nxiyi−(∑i=1nxi)(∑i=1nyi)n∑i=1nxi2−(∑i=1nxi)2

其中 y^ 为缺失值填补结果，x为缺失值对应的年份；若 y^<0 （非负指标），则用0填补。

3. 数据归一化

为消除量纲差异，将所有指标统一转化为0-1区间，且“越大越好”：

  • 正向指标（越大越好）： Ii,t,c′=Ii,t,c−min(Ii,..)max(Ii,..)−min(Ii,..)
  • 负向指标（越小越好）： Ii,t,c′=max(Ii,..)−Ii,t,cmax(Ii,..)−min(Ii,..)

三、三大核心模型详解

模型1：关系图模型（Relationship Graph Model）

核心目的：量化17个SDGs之间的协同/权衡关系，为优先级排序提供基础指标。

1. 图结构构建

用无向图 G(V,E,W) 表示SDGs关系：

  • 节点 Vi ：第 i 个SDG目标；
  • 边 Eij ：目标 i 与 j 的关联（用Spearman相关系数衡量）；
  • 权重 wij ：关联强度（正向=协同，负向=权衡）。

2. 关键算法：Spearman相关分析

选择Spearman而非Pearson的原因：更适合捕捉非线性相关，对异常值不敏感。

  • 步骤1：将序列数据转化为等级向量 rik （排序后的位置）；
  • 步骤2：计算序列间相关系数：$$b_{ik,jp}=\frac{\frac{1}{n} \sum_{m=1}^{n}(r_{ikm}-\overline{r_{ik}}) \cdot (r_{jpm}-\overline{r_{jp}})}{\sqrt{(\frac{1}{n} \sum_{m=1}^{n}(r_{ikm}-\overline{r_{ik}})^2) \cdot (\frac{1}{n} \sum_{m=1}^{n}(r_{jpm}-\overline{r_{jp}})^2)}}$$
  • 步骤3：分类统计关联类型（ b>0.6 为正相关， b<−0.6 为负相关， |b|≤0.6 为无相关）；
  • 步骤4：计算目标间权重： wij=aij1−aij2 （ aij1 为正相关比例， aij2 为负相关比例）。

3. 图度量：节点重要性指标

用3类中心性（分别计算正向边/负向边）量化目标的“网络影响力”，共6个指标：

  • 介数中心性（BC）： BC(i)=∑j<kj≠i≠kni(j,k)n(j,k) （衡量节点的“桥梁作用”）；
  • 接近中心性（CC）： CC(i)=1∑j≠id(i,j) （衡量节点的“可达性”）；
  • 特征向量中心性（EC）： EC(i)=α∑j=1nrij⋅EC(j) （衡量节点的“核心辐射力”）。

4. 核心发现

  • 多数SDGs呈正相关（协同关系），符合联合国设定目标的一致性；
  • Goal7（廉价清洁能源）与所有目标呈负相关（技术投入大、资源竞争激烈）。

模型2：优先级模型（Priority Model）

核心目的：基于关系图模型的6个中心性指标，客观排序17个SDGs的优先级。

1. 方法组合：熵权法（EWM）+ 加权TOPSIS

  • 熵权法：确定6个指标的客观权重（避免主观偏差）；
  • 加权TOPSIS：基于权重计算目标与“最优解”“最劣解”的距离，排序优先级。

2. 熵权法步骤

  1. 指标归一化： z~ij=cij−min{c1j,c2j,...,cnj}max{c1j,c2j,...,cnj}−min{c1j,c2j,...,cnj} ；
  2. 计算概率： pij=z~ij∑i=117z~ij ；
  3. 信息熵： ej=−1ln17∑i=117pijln(pij) ；
  4. 权重计算： wj=1−ej∑j=16(1−ej) 。

3. 加权TOPSIS步骤

  1. 标准化矩阵： zij=cij∑i=117cij2 ；
  2. 定义最优/最劣向量： z+ （各指标最大值）、 z− （各指标最小值）；
  3. 加权距离计算：
    • 到最优解距离： Di+=∑j=16wj(zj+−cij)2 ；
    • 到最劣解距离： Di−=∑j=16wj(zj−−cij)2 ；
  4. 优先级得分： si=Di−Di++Di− （得分越高，优先级越高）。

4. 核心发现

  • 普适性结论：Goal3（健康）和Goal4（教育）在三个国家均进入前五，是全球普适性高优先级目标；
  • 国别差异：土耳其优先Goal17（伙伴关系）和Goal9（产业创新），印尼/墨西哥优先Goal3（健康）。

模型3：时间香槟塔模型（Temporal Champagne Tower Model）

核心目的：基于优先级和SDGs关联，预测未来10年发展进度（资源动态分配）。

1. 设计灵感

香槟塔的“上层满溢后流入下层”——资源优先分配给高优先级目标，完成后资源流向关联目标，同时维护已完成目标。

2. 核心假设

所有SDGs的“完成难度相同”（1%完成度的资源投入等价），简化量化复杂度。

3. 关键变量与公式

  • 完成率： Gt=[gt(1),gt(2),...,gt(17)] （ gt(i) 为第 t 年第 i 个目标的完成百分比）；
  • 发展队列： St+1=St∪St′ （ St′ 为与当前队列关联的目标集合）；
  • 协同影响： vt(i)=1C∑jVj∈Stwij(vt−1(j)+dt−1(j)) （ C=5 基于帕累托原则）；
  • 资源分配： dt(i)={0,ifvt(i)+gt−1(i)≥1min(1−vt(i)−gt−1(i),D−∑s(j)>s(i)max(0,1−vt(j)−gt−1(j))),else ；
  • 完成率更新： gt(i)=min(gt−1(i)+dt(i)+vt(i),1) 。

4. 核心结果（以印尼为例）

  • 无突发事件：2033年完成5个目标（Goal3/4/6/9/13），整体完成率82%（比当前计划高4%）；
  • 创新建议：新增Goal18（国际合作），利用已完成目标的协同效应帮助其他国家。

四、扩展分析：突发事件与敏感性检验

1. 突发事件影响分析（COVID-19为例）

  • 新增参数：威胁系数 γi （衡量目标受疫情威胁程度）、缓解/加剧系数 MIi （衡量目标对疫情的影响）；
  • 模型调整：资源分配公式加入 γi ，优先级指标加入 MIi ；
  • 结果：印尼2033年仅完成3个目标，整体完成率78%（比无疫情低4%）。

2. 敏感性分析（验证鲁棒性）

  • 分析对象：超参数 C （协同效率常数）和 D （年总资源）；
  • 测试范围： C 取4-6（步长0.1）， D 取0.102-0.136；
  • 结果： C 的最大影响幅度0.041， D 的最大影响幅度0.028，模型对超参数变化不敏感，鲁棒性强。

五、模型亮点与不足

1. 亮点

  • 全面性：兼顾SDGs的正向/负向关联、静态优先级/动态发展、多场景应用；
  • 创新性：香槟塔模型将“资源分配+时间维度+关联溢出”结合，隐喻直观且数学表达严谨；
  • 实用性：为联合国、企业提供具体可操作的建议。

2. 不足

  • 超参数估计依赖专家数据（如疫情相关参数 γi 和 MIi ）；
  • 假设简化：忽略SDGs的完成难度差异。

六、应用价值

  1. 对国家：健康（Goal3）和教育（Goal4）是普适性高优先级目标，可根据自身情况调整目标优先级；
  2. 对企业：小企业聚焦国家高优先级目标（易获投资），大企业可利用模型分析SDGs协同关系，制定长期战略；
  3. 对联合国：建议降低发展中国家Goal7/11/17的标准，推动国际合作，适当延长SDGs完成期限。
